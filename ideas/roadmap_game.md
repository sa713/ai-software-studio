ROADMAP_GAME_DESIGNER_V01

Цель

Добавить в Студию роль Game Designer.

Роль должна участвовать в разработке игровых продуктов и выполнять независимые Game Design Review.

⸻

Stage 1. Определение зоны ответственности

Game Designer отвечает за:

* core loop;
* progression;
* rewards;
* pacing;
* retention;
* player motivation;
* meaningful choices;
* risk/reward;
* game economy на концептуальном уровне.

Не отвечает за:

* UI layout;
* навигацию;
* визуальный дизайн;
* архитектуру;
* код.

⸻

Stage 2. Встраивание в производственный цикл

Новая цепочка для game-проектов:

Product Analyst

↓

UX Designer

↓

Product Owner

↓

Game Designer

↓

Solution Architect

Game Designer получает уже принятые продуктовые цели и превращает их в игровые системы.

⸻

Stage 3. Game Design Artifacts

Добавить:

* GAME_DESIGN_REVIEW.md
* GAME_DESIGN_NOTES.md
* GAMEPLAY_RISKS.md
* GAMEPLAY_OPPORTUNITIES.md

⸻

Stage 4. Game Design Review Mode

Режим:

Game Design Review

Проверяет:

* интересно ли играть;
* есть ли доминирующая стратегия;
* есть ли ложные выборы;
* есть ли скучные участки;
* работает ли progression;
* работает ли retention.

⸻

Stage 5. Gameplay Findings

Ввести новый Source Of Work:

Gameplay Finding

Примеры:

* Proof не мотивирует;
* Research воспринимается как обязанность;
* игрок не понимает ценность системы;
* существует доминирующая стратегия.

После Product Owner Review может попасть в backlog.

⸻

Stage 6. Balance & Progression Reviews

Добавить специальные ревью:

* Core Loop Review
* Progression Review
* Reward Review
* Retention Review

Эти ревью могут автоматически вызываться Mission Mode.

⸻

Stage 7. Mission Mode Integration

Mission может автоматически запускать:

Game Design Review

после:

* завершения roadmap stage;
* крупных изменений экономики;
* крупных изменений progression.

Результат:

Mission получает обратную связь не только по качеству реализации, но и по качеству игрового опыта.

⸻

Критерий завершения

Game Designer:

* встроен в основной цикл игровых проектов;
* умеет делать независимый Game Design Review;
* создаёт Gameplay Findings;
* участвует в Mission Mode;
* способен генерировать Source Of Work на основании проблем игрового опыта.
