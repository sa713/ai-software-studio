ROADMAP_UX_DESIGNER_V01

Цель

Добавить в Студию роль UX Designer.

Роль должна уметь:

* участвовать в основном производственном процессе;
* выполнять самостоятельные UX Review;
* формировать UX Source Of Work;
* влиять на backlog через Product Owner и Delivery Planner.

UX Designer отвечает за удобство использования продукта.

Не отвечает за игровые механики.

Не отвечает за бизнес-логику.

Не отвечает за архитектуру.

⸻

Stage 1. Определение зоны ответственности

Закрепить, что UX Designer отвечает за:

* информационную архитектуру;
* навигацию;
* onboarding;
* discoverability;
* когнитивную нагрузку;
* читаемость интерфейса;
* структуру экранов;
* пользовательские сценарии.

Не отвечает за:

* баланс;
* core loop;
* rewards;
* retention;
* экономику игры.

Результат:

UX Designer имеет уникальную зону ответственности без пересечения с Product Analyst.

⸻

Stage 2. Встраивание в производственный цикл

Новая цепочка:

Product Analyst

↓

UX Designer

↓

Product Owner

↓

Solution Architect

UX Designer получает результаты анализа и создаёт:

* UX Notes;
* UX Risks;
* UX Requirements;
* UX Recommendations.

Результат:

проблемы интерфейса выявляются до реализации.

⸻

Stage 3. UX Artifacts

Добавить артефакты:

* UX_REVIEW.md
* UX_REQUIREMENTS.md
* UX_RISKS.md

Зафиксировать их место в lifecycle.

⸻

Stage 4. UX Review Mode

Добавить отдельный режим:

UX Review

Вход:

* проект;
* экран;
* сценарий;
* приложение.

Выход:

* UX Findings;
* Severity;
* Recommendations.

Результат:

UX Designer может работать как независимый ревьюер.

⸻

Stage 5. UX Source Of Work

Разрешить UX Designer создавать:

UX Finding

UX Finding становится допустимым Source Of Work.

После Product Owner Review может попадать в backlog.

⸻

Stage 6. Mission Mode Integration

Mission может автоматически вызывать UX Review:

* после milestone;
* после релиза;
* после завершения roadmap stage.

Результат:

UX начинает участвовать в автономных миссиях.

⸻

Критерий завершения

UX Designer:

* встроен в основной цикл;
* умеет делать UX Review;
* создаёт UX Source Of Work;
* участвует в Mission Mode.
